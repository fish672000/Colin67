Le Web

Le Web ou World Wide Web est un sytème hypertexte fonctionnant sur Internet. L’hypertexte est le fondement de l’architecture du web. Ce concept a été inventé en 1965 par Ted Nelson. Un lien hypertexte est un élément d’une page web sur lequel on peut cliquer pour accéder à une autre page web. Ces liens servent donc à relier les pages web entre elles et permettent de naviguer de l’une à l’autre. Ces pages web sont repérées par des adresses URL (Uniform Ressource Locator). Pour accéder à ces pages web, il est nécessaire de passer par un navigateur et de formuler une requête sur ce navigateur selon un certain protocole (http ou https en général).

Quelques dates clées:

1965: Ted Nelson un sociologue américain imagine l'accèes rapide à des blocs d'information contenu dans des documents selon l'intérêt du lecteur. C'est ainsi que le lien hypertexte est né.

1989: Tim Berners-Lee un informaticien britannique, définit dans un laboratoire du CERN, les principes de ce qui deviendra le Web et réalise un premier prototype de serveur, ainsi qu’un programme client.

1990: Premier site web: The Project

1992: Différents termes apparaissent: HTML: langage conçu pour représenter une page web, URL: adresse pour identifier une page web et HTTP: protocole de communication entre l’utilisateur et le serveur détenant la page web

1993: Mosaic : navigateur web compatible sous Windows, Linux et Mac OS.

1994:W3C, World Wide Web Consortium, uniformisation des standards Première transaction sécurisée en ligne. Premier annuaire de sites:Yahoo.

1995:Les pages web deviennent dynamiques grâce à deux langages JavaScript et PHP.

2005:Emergence du Web 2.0:les internautes ne se contentent plus de consommer des contenus, ils en produisent et les partagent via de nouvelles plateformes communautaires.

2008: Arrivée des applis mobiles.
